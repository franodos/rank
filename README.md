## 排行榜服务
请求Content-Type设置为application/json
### 接口1：上次客户端编号及分数
- url
```
/client/
```
- body提交的数据
```
number: int。客户端编号，大于等于0
grade: int。分数，分数范围为1...10000000
```
- 返回值示例：
```
{
    "id": 9,
    "number": 6,
    "grade": 200
}
```

### 接口2：获取排行榜
- url
```
/rank/
```
- 参数
```
number: int。 客户端编号
start: int, 默认为0。排行榜起始位置，大于等于0
stop: int， 默认为-1。排行榜结束位置，大于等于0或-1
```
- 返回值示例：
```
# http://127.0.0.1:8000/rank/?number=6&start=0&stop=-1
[
    [
        6,
        200
    ],
    [
        5,
        200
    ],
    [
        3,
        102
    ],
    [
        2,
        100
    ],
    [
        1,
        100
    ],
    [
        6,
        200
    ]
]
```
